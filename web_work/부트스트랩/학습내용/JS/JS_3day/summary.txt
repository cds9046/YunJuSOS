* for 문의 기본구조
 - for( 초기식; 조건식; 증감식; ) { 실행문; }


* continue문
 - 반복문과 함께 사용
 - 반복문 안에서 continue문을 만나면 이후 문장은 실행하지 않고 for문의 증감식으로 이동한다
 - Jump 라고 생각하면 쉽다!


* 중첩 for문
 - for문 안에 for문 사용
 - for( 초기식; 조건식; 증감식; ) {
     for( 초기식; 조건식; 증감식; ) {
        실행문;
        }
    }


* while 문
 - 반복횟수를 정하지 않고 특정 조건이 주어질 경우, 반복문이 종료
 - while(조건) {
    실행문;
    }
