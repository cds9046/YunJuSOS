 떡볶이 

 재료


 떡볶이용 떡 500g, 어묵 150g (사각형 어묵 3장), 물 한컵 반(360ml)

 양념장 : 고추장 3큰술, 간장 1/2큰술, 설탕 2큰술, 물엿 1큰술

 준비하기


 1. 떡볶이용 떡은 말랑하면 그냥 씻어 건져두고, 단단히 굳었으면 끓는 물에 살짝 데쳐서 사용한다.

 2. 어묵은 2.5 * 5cm 크기로 썰어 체에 올려놓고 뜨거운 물을  부어 기름을 뺀다.

 만들기


 1. 팬에 물을 넣고 고추장, 간장, 설탕, 물엿을 넣어 잘 푼다.

 2. 1의 팬을 불에 올리고 떡을 넣어 눌러붙지 않게 저어가며     잠시 끓인다.

 3. 어묵을 넣고 양념장이 걸쭉해질 때까지 끓인다.
