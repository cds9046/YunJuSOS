* 입력 양식 태그 (form, input 등)
 2. <input> 태그
  - 사용자의 입력값을 받기 위한 태그
  - 사용자의 입력값은 모두 문자열로 취급 (산술 연산 불가)
   : 산술 연산을 할 경우 숫자로 변환해야 함
  - 속성
   1) type : input 태그의 모양을 결정하는 속성
   2) name : 전달할 데이터에 붙일 이름을 지정하는 속성
   3) value : 전달할 데이터가 저장되는 속성
            : 또는 버튼 type의 input 태그일 경우, 출력하는 글자를 작성하는 속성
              (버튼 type의 input은 value를 전달하지 않는다)
      참고) key-value 저장방식
          : 키 값과 데이터를 같이 저장하여 데이터를 구할 때 키 값을 사용하는 저장방식
      참고) type 속성의 값
          1) submit : 버튼. form 태그의 action 속성에 지정된 대상으로
                     input 태그의 name과 value를 전송
          2) reset : 버튼. 한 form 태그에 소속된 모든 input 태그의 value 초기화
          3) text : 문자열 입력형식 (기본형태)
          4) password : 문자열 입력형식. 입력문자를 자동으로 '●'로 변경
          5) hidden : 화면에 보이지 않는 형식
          6) button : 버튼. 자바스크립트 코드와 연결하여 이벤트를 처리하는 형식
          7) radio : 단일 선택용 입력형식 (Single-choice)
                   : 여러 요소 중 하나를 선택하는 용도로 사용
                   : name 속성에 같은 값을 넣어 하나로 그룹화 함
          8) checkbox : 다중 선택용 입력형식 (Multiple-choice)
                      : 여러 요소 중 여러개를 선택하는 용도로 사용
                      : name 속성에 같은 값을 널어서 하나로 그룹화 함
          9) date : 년-월-일 단위로 날짜를 입력하는 형태
          10) month : 년-월 단위로 날짜를 입력하는 형태
          11) week : 년-주 단위로 날짜를 입력하는 형태
          12) time : 시간 입력 형태
          13) datetime-local : 년-월-일-시 단위로 입력하는 형태
          14) color : 색상 선택창을 출력하여 원하는 색상을 선택/입력
          15) number : 숫자를 입력하는 형태 (숫자 문자열임!!!)
                      (min, max로 하한/상한 설정)
          16) range : 특정 범위의 숫자를 선택하여 입력하는 형태
                      (min, max로 하한/상한 설정)
          17) file : 업로드용 파일을 선택할 수 있는 형태. 파일 선택 창이 뜸
                     (다중 선택 시 Multiple 사용)
          18) email : 이메일 형식으로 입력을 제한하는 형태
          19) url : 사이트 주소 형식으로 입력을 제한하는 형태
          20) tel : 전화번호 형식으로 입력을 제한하는 형태
                    (pattern 속성으로 형식을 지정 [pattern="\d{2,3}-\d{3,4}-\d{4}"])
  - input 태그의 주요 속성
   1) readonly : 읽기 전용으로 수정할 수 없도록 지정하는 속성 (반드시 value 값을 넣을 것!)
   2) disabled : 요소를 비활성화 상태로 변경하는 속성 (submit로 전송되지 않는다!)
   3) size : text 타입의 input 태그의 가로 길이(너비)를 지정.
             입력 문자를 제한하지 않음 (ex. size="20" <- 20글자가 들어가는 공간너비)
   4) maxlength : 입력 문자의 개수를 제한하는 속성
 ★5) placeholder : 힌트 글을 출력하는 속성
 ★6) required : input 요소를 필수 입력 요소로 만드는 속성
   7) autofocus : 페이지가 열릴 때(or 새로고침) 처음으로 입력받는 input 요소로 지정하는 속성

* 기타 입력 태그 (와 관련 태그들)
 1) textarea : 여러 줄의 문장(문자열)을 입력받는 요소.
             : 크기지정
              1) rows - 세로 길이(줄 수)
              2) cols - 가로 길이(글자 수)
              시작 태그와 종료 태그 사이의 값을 전송 (value 안 씀)
 2) select : 선택 목록을 출력하는 요소 (name은 select에 작성)
           : <option> 태그와 같이 사용 <- 항목 출력 태그
            ( <ul> 또는 <ol> 또는 <li> 역할을 담당하는 태그)
           : <option> 태그의 value 값이 같이 전송된다
 3) datalist : input + select 요소
             : 일종의 자동완성 기능처럼 동작
             : input 태그에 list 속성을 사용.
               datalist에 id 속성을 사용.
               list 속성값과 id 속성값은 같아야 한다.
 4) label : 특정요소의 레이블을 정의하는 요소
          : 요소의 선택(클릭) 영역을 확장하는데 사용
          : label 태그의 for 속성값과 input 태그의 id 속성값이 같으면 두 요소가 연결됨

      참고) 이벤트(Event)란?
          : 컴퓨터에 의해 감지되고 처리될 수 있는 동작이나 사건
            프로그램은 사용자가 발생시킨 이벤트를 처리하는 명령어 묶음
            이벤트를 처리하는 작업을 '이벤트 핸들링(Event Handling)'이라 함
          : 마우스클릭, 마우스무브, 키보드눌림 등

      참고) 버튼 생성 방법
          1) input 태그의 type을 button으로 하여 생성
          2) button 태그로 생성. form 태그 안에 button태그로 만들 버튼은
             input 태그 중 type이 submit인 버튼의 동작을 수행하기에,
             type="button"을 넣어준다

  ※참고) 데이터 형태
   1) 숫자(Number) : 정수(-100,38 등), 실수(3.14 등)
   2) 문자(Character) : 알파벳, 한글 등 다국의 언어
   3) 문자열(String) : 문자의 배열 (단어, 문장)
   4) 불리언(Boolean) : 진리값. True(0 아닌 값) / False(0)
